# GymBuddy

GymBuddy is a fitness-focused social networking app that connects people looking for workout partners. Built with React Native (Expo) for iOS and Android.

## MVP Features
- User authentication and profile creation
- Swipe-based matching
- Real-time chat (1:1 and group)
- Gym check-in and verification
- Workout session scheduling
- Progress sharing
- Integration with Firebase backend

## Getting Started

### Prerequisites
- Node.js
- npm or yarn
- Expo CLI (`npm install -g expo-cli`)

### Running the App
1. Install dependencies:
   ```
   npm install
   ```
2. Start the development server:
   ```
   npm start
   ```
3. Use the Expo Go app on your device or an emulator to preview.

## Folder Structure
- `src/features/auth` - Authentication
- `src/features/profile` - User profiles
- `src/features/match` - Matching system
- `src/features/chat` - Messaging
- `src/features/checkin` - Gym check-in
- `src/components` - Shared UI components
